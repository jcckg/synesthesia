# Font Assets

## Font Awesome 6 (Required for Icons)

Download the Font Awesome 6 Free font file:

1. Visit: https://github.com/FortAwesome/Font-Awesome/blob/6.x/webfonts/fa-solid-900.ttf
2. Click "Download" or "Raw" to download the file
3. Save as `fa-solid-900.ttf` in this directory

Alternatively, download from the official Font Awesome website:
- https://fontawesome.com/download (Free for desktop)
- Extract and copy `webfonts/fa-solid-900.ttf` to this directory

The application will look for this font file at runtime to display icons in the UI.
